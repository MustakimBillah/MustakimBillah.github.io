Telenor Maritime Service Hub

December 2019 - Present
ASP.NET Core, Vue Js, Couch Db, Graph CMS, Mapbox GL JS, Cent Os

About

Service Hub is a solution for ship travellers which enable them to access wifi from ship server and enjoy a lot of interesting stuffs like watching movies & tv series, view ship moving on map and explore the facilities of the ship from the developed website.

Backend & Frontend

We have used ASP.NET CORE as backend technology and generated apis, then we chose VueJs as our frontend technology to show user an interactive layout.

GRAPHCMS

To maintain cms functionality we have used GRAPHCMS which uses graphQL as query language. First we create contents like ship,destination etc and then we fetch it on backend dock server. After that we push the data to several backend ship servers.

CouchDB & CentOS

For replicating database from dock server to ship servers we have used CouchDB. Also we have used CentOS on our servers.
